Mercado-Escolar:
----------------------------------------
System requirements:
laravel 5.0 or above
reactjs
Javascript
html
css
PHP 8.1.0 or higher
mysql 9.0 or higher
composer latest version
----------------------------------------------
-> all the files for the project are present in the project marketplace zip folder
-> extract the folder
-> open terminal
run the following command

npm install
npm run dev
-> open another terminal and go the folder path and enter

composer install
composer update
php artisan migrate
php artisan serve

-> once done the project will be loaded in laravel and you can access using 

http://127.0.0.1:8000/
